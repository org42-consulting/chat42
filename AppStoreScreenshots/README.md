# App Store Screenshots

This directory contains the screenshots and assets needed for App Store submission.

## Screenshot Requirements

For macOS App Store, the following screenshot sizes are required:

### Main Screenshots (4)
- 1280×800 pixels (for Mac App Store)
- 1920×1200 pixels (for Mac App Store)
- 2560×1600 pixels (for Mac App Store)
- 3840×2400 pixels (for Mac App Store)

### Additional Assets
- App icon (1024×1024 pixels)
- Marketing materials
- Promo graphics

## File Naming Convention
- `screenshot_1280x800.png`
- `screenshot_1920x1200.png`
- `screenshot_2560x1600.png`
- `screenshot_3840x2400.png`

## Submission Notes
- All screenshots should be in PNG format
- Screenshots should show the app in action
- Include both light and dark mode screenshots where applicable
- Ensure text is readable and clear
- Follow Apple's App Store guidelines for content and presentation
